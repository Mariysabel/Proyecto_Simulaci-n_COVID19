# Proyecto de simulación de la evolución del COVID-19

Este trabajo presenta un modelo de simulación computacional para analizar la evolución de los casos de COVID-19 en el estado de Oaxaca. A partir de datos oficiales, se modelan los procesos de contagio, recuperación y defunción como parte de un sistema dinámico de salud pública. Los resultados ofrecen una herramienta útil para predecir tendencias y apoyar la toma de decisiones en materia de salud pública.

## Comandos disponibles

En este proyecto puedes ejecutar:

### `npm start`

Ejecuta la aplicación en modo desarrollo.
Abre http://localhost:3000 en tu navegador.
La página se recarga al hacer cambios y puedes ver errores en la consola.

### `npm run build`

Crea una versión optimizada para producción en la carpeta build.
Los archivos se minifican y están listos para desplegar.
Más info en la sección de deployment.
